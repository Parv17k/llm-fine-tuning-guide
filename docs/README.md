# LLM Fine-Tuning Guide

> A Step-by-Step Guide for Technical People

No ML background required. Learn to fine-tune LLMs from scratch.

---

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Parv17k/llm-fine-tuning-guide.git
cd llm-fine-tuning-guide

# Set up environment
pip install -r requirements.txt
```
